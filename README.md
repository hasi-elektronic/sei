# 清 SEI — Intermittent Fasting Tracker

> Klar. Rein. Bewusst.

**SEI** is a Zen-inspired IF tracking app for people who want clarity.

## Features
- ⏱ Fasting timer (20:4)
- 🍽 Meal tracker with calorie + protein calc
- ⚖ Weight log with chart
- 💧 Water intake
- 📊 BMR/TDEE calculations
- 🔐 Auth (register/login)

## Stack
- Frontend: React 18 + TypeScript + Vite + Tailwind
- Backend: Cloudflare Workers + TypeScript
- DB: Cloudflare D1

## Deploy
- Frontend → sei.pages.dev (GitHub Actions auto-deploy)
- Backend → Cloudflare Workers
